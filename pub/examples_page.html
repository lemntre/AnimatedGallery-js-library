Examples Documentation
Examples

This page displays the five types of galleries available, grid, line up, diamond, panel, and chessboard.

Click each image to see it in detail with its title and description.

Hover over the dropdown menus above each gallery to change the according style.

Sorting refers to the order the images are sorted by. There are 3 orderings available, alphabetical, clicks, and as input.

Alphabetical: alphabetical by title of the image.

Clicks: descending order by number of times users have clicked on the image, all images start with a times clicked of zero. To test this sorting click into one of the images and sort.

As input: sorted by the order the images were inputted in. The default sorting for all galleries on this page is as input.

Hover Animation refers to the animation triggered by hovering over that image, this animation applies to individual images. There are at most 3 hover animations available, scale up, scale down, and drop shadow.

Animation refers to the animation that applies to all the images when the gallery is initially displayed. There are 7 animations available. fade in, slide up, slide down, slide left, slide right, slide cross vertically, slide cross horizontally.

To change the speed of the animation, input a value into the input field, example: '3s', then clicking the set button will change the speed of the animation. If the input is invalid, nothing will happen.

Scale Down Scale Up Drop Shadow
Fade In Slide Up Slide Down Slide Left Slide Right Slide Cross Vertical Slide Cross Horizontal None

const ag1 = new AnimatedGallery('gallery1')


ag1.setStyle('grid')
ag1.setAnimation('none')
ag1.setSorting('asInput')
ag1.setElements(
