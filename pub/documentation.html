Getting Started

Include AnimatedGallery.js and AnimatedGallery.css in your current directory, import the package and jQuery using the following script:

<link rel="stylesheet" type="text/css" href="AnimatedGallery.css">
<script defer type="text/javascript" src="AnimatedGallery.js"></script>
<script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

API
Creating a New AnimatedGallery: new AnimatedGallery(containerID)

To create a new AnimatedGallery instance, pass in the id of the parent container that should contain your AnimatedGallery.

<div id="myGalleryID"></div>
const myGallery = new AnimatedGallery("myGalleryID")

Setting Style: .setStyle(styleName)

Sets the style of the gallery, valid inputs for styleName are gird, line up, diamond, panel, chessboard. See examples page for samples.

myGallery.setStyle('grid')
myGallery.setStyle('line up')
myGallery.setStyle('diamond')
myGallery.setStyle('panel')
myGallery.setStyle('chessboard')

Setting the Gallery Elements: .setElements(myElements)

Sets the elements in the gallery. Valid input is an array, each element of the array must have: a src attribute that is the source of the thumbnail image; a title attribute that is the title of the image that will be displayed in the lightbox; a description attribute that is the description of the image that will be displayed in the lightbox. An element can also have an optional index attribute which represents the order that the elements will be displayed in, the default index value of each element is its position in the elements array.

const myElements = [
   {src: 'image1.jpg', title:'image 1', description: 'This is the first image'}
   {src: 'image2.jpg', title:'image 2', description: 'This is the second image', index: 2}]
myGallery.setElements(myElements)

Making the Gallery: .makeGallery()

Makes the gallery with the given style and elements.

myGallery.makeGallery()

Setting Animation: .setAnimation(animationName)

Sets the Animation of the gallery that applies to every thumbnail image. Valid inputs for animationName are fade in, slide up, slide down, slide left, side right, slide cross v, slide cross h, none. See examples page for samples.

myGallery.setAnimation('fade in')
myGallery.setAnimation('slide up')
myGallery.setAnimation('slide down')
myGallery.setAnimation('slide left')
myGallery.setAnimation('slide right')
myGallery.setAnimation('slide cross v')
myGallery.setAnimation('slide cross h')
myGallery.setAnimation('none')

Setting Sorting: .setSorting(sortingName)

Sets the order the elements are sorted by. Valid inputs for sortingName are alphabetical, clicks, asInput. See examples page for samples.

myGallery.setSorting('alphabetical')
myGallery.setSorting('clicks')
myGallery.setSorting('asInput')

Setting the Hover Animation: .setHoverAnimation(hoverAnimation)

Sets the hover animation that applies to each thumbnail image. Valid inputs for hoverAnimation are scale up, scale down, drop shadow.

myGallery.setHoverAnimation('scale up')
myGallery.setHoverAnimation('scale down')
myGallery.setHoverAnimation('drop shadow')

Setting the Animation Duration: .setAnimationDuration(duration)

Sets the duration of the animation that applies to all thumbnail images. Valid inputs for duration is a string that is a valid time value. See examples page for samples.

myGallery.setAnimationDuration('5s')

Changing the Gallery Style: .newStyle(newStyle)

Changes the style of an existing gallery. Valid inputs for newStyle are the same as styleName from the .setAnimation() function description.

myGallery.newStyle('line up')

Changing the Gallery Animation: .newAnimation(newAnimation)

Changes the animation of an exisiting gallery. Valid inputs for newAnimation are the same as animationName from the .setAnimation() function description.

myGallery.newAnimation('slide right')

Changing the Gallery Sorting: .sortElements(newSorting)

Changes the sorting order of an existing gallery. Valid inputs for newSorting are the same as sortingName from the .setSorting() function description.

myGallery.sortElements('alphabetical')

Adding Elements: .addNewElement(newElement)

Adds a new element to an existing gallery.

myGallery.addNewElement(
   {src:'image3.jpg', title:'image 3', description:'This is the third image.'})
